黑盒测试呢是大家在工作中最常进入的工作岗位，就是指呢我们公司会接到一个项目，就像互网一样，让你去渗透一个公司。现在只告诉你这个公司的名字，或者是只告诉你这个公司的官方网站，让你去渗透。

除了公司的网站之外，所有的信息都需要你自行收集，所有的漏洞都需要你自行扫描或者是挖掘。这个叫做黑盒测试，就像摸黑一样，自己去探索。第二种呢叫做白盒测试。白盒测试这工作岗位我们经常遇到的叫做代码审计。
